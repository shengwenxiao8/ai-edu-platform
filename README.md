# AI Education Platform

> 🚧 Under Development

An AI-powered personalized learning platform that combines intelligent tutoring agents,
automated assessment, and RAG-based knowledge Q&A to deliver adaptive learning experiences.

## Features (Planned)

- [ ] AI-generated course content and curriculum
- [ ] Intelligent tutoring agent (personalized learning paths)
- [ ] Smart quiz generation and auto-grading
- [ ] RAG-powered knowledge base Q&A
- [ ] Learning progress tracking and analytics
- [ ] Multi-agent collaboration (Tutor + Quiz + Evaluator)

## Tech Stack

- **Frontend:** Next.js / React / TailwindCSS / shadcn/ui
- **Backend:** Python / FastAPI
- **AI:** Claude API / Agent SDK / RAG Pipeline
- **Database:** PostgreSQL + Chroma (vector)
- **Deployment:** Docker / Vercel

## Status

🔜 Development starts in Phase 3 (W17-W20)
